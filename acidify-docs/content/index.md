# 快速开始

欢迎来到 Acidify 的文档！

`acidify-core` 是一个用 Kotlin 编写的跨平台 PC & Android NTQQ 协议支持库，发布于 Maven Central 和 npm。Acidify 的代码仓库还包含一个基于 `acidify-core` 实现的 QQ 协议端 Yogurt。

## 使用

### 通过 Milky 协议对接其他框架

[点击这里](./yogurt/start.md)了解如何下载和使用 Yogurt。

### 基于 Acidify 进行开发

了解 `acidify-core` 如何[在 Kotlin 中使用](./core/kotlin.md)。

更喜欢 JavaScript/TypeScript？请查看[在 Node.js 中使用](./core/nodejs.md)。
